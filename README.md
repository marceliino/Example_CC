# Example_CC

A hands-on example project for exploring [Claude Code](https://claude.ai/code) — Anthropic's official CLI and agentic coding assistant.

## About

This repository serves as a sandbox for learning and experimenting with Claude Code features, including:

- Agentic code generation and editing
- Multi-file refactoring
- Git workflow automation
- MCP (Model Context Protocol) server integrations
- Custom hooks and settings

## Getting Started

### Prerequisites

- [Claude Code](https://claude.ai/code) installed
- Node.js 18+ or Python 3.9+ (depending on the examples you run)
- Git

### Installation

```bash
git clone https://github.com/marceliino/Example_CC.git
cd Example_CC
```

### Running Claude Code

```bash
# Launch Claude Code in the project directory
claude
```
